# Stock Info Notification LINE Bot

## Overview
A Node.js application that periodically fetches stock information recommended by Mr.KIRITANI, a well-known stock trader, from the latest YouTube videos and web articles, then sends messages generated by Gemini AI to the LINE Bot

## LINE Account
https://line.me/R/ti/p/@355pxzmn
(You need to have your own LINE account to follow this)

## Technologies
- TypeScript
- Node.js
- tsyringe (DI container)
- node-cron (scheduler)
- Gemini API

## How to Run
1. Install required packages
   ```
   npm install
   ```
2. Run the application
   ```
   npm run dev
   ```

## Notes
- The information fetch task runs automatically every day at 9:00 AM.
